# [InputNumber](https://greasyfork.org/zh-CN/scripts/432807)

自定义数值输入框元素。

```html
<input is="laster2800-input-number" value="5.6" min="-2.8" max="10" digits="2">
```

使用 `min` / `max` 限制范围，`digits` 设置精度。可使用上下方向键（配合 Alt/Shift/Ctrl）调整数值大小。
